## Getting Started

1. Clone the repo.

```
git clone https://github.com/kaiwangm/Engine.git
```

2. install dependencies.

```
git clone https://github.com/microsoft/vcpkg
.\vcpkg\bootstrap-vcpkg.bat
```

3. Build.

```
./Script/build_windows.bat
```
